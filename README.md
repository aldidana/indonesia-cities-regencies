# List of all cities/regencies in Indonesia

## Install

```javascript
npm install indonesia-cities-regencies --save
```

## Example

```javascript
const indonesia = require('indonesia-cities-regencies')

indonesia.get('jak') // default limit is 10
indonesia.get('jak', 5)
```
